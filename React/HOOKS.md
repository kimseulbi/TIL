# HOOKS

class를 작성하지 않고도 state와 다른 React의 기능들을 사용할수 있게 해줍니다. -> function

### HOOKS의 특징

- 당신은 현재 코드를 다시 쓸 필요 없이 몇몇 컴포넌트들에 Hooks를 사용할 수 있습니다. (새로운 선택권 ㅋㅋ)
- 100% 이전 버전과의 호환성 : Hooks는 어떠한 변화도 포함하지 않습니다.
- Hooks는 현재 버전 16.8.0과 함께 이용이 가능
